Skeptic Privacy Policy

Last updated: March 15, 2026

The short version: Skeptic does not collect, store, or transmit your personal data. Everything stays on your device.

What Skeptic Does

Skeptic is a weather app that compares forecasts from multiple sources and tracks their accuracy over time. It uses your device's location to fetch weather data and displays it locally.

Location Data

Skeptic requests your location to look up weather forecasts for where you are. Your location is:

  • Used only on your device to make weather API requests
  • Never sent to HogBody Labs
  • Never stored on any server
  • Never shared with anyone

You can deny location access and manually add locations instead.

Weather Data Sources

When you use Skeptic, your device contacts these weather services directly to fetch forecasts:

  • Open-Meteo (no API key required)
  • National Weather Service / NWS (no API key required)
  • Apple WeatherKit
  • MET Norway (no API key required)
  • OpenWeatherMap (your API key)
  • WeatherAPI.com (your API key)
  • Pirate Weather (your API key)

For services that require API keys, you provide your own key in Settings. Skeptic stores these keys securely in your device's Keychain. HogBody Labs never sees your API keys.

Sensor Integrations

Skeptic can optionally connect to personal weather sensors:

  • Govee — WiFi thermometer/hygrometer (your API key)
  • Davis WeatherLink — Personal weather station (your API key and secret)

These connections are optional. If configured, your device contacts these services directly using credentials you provide. HogBody Labs never receives sensor data or API credentials.

Data Storage

All app data is stored locally on your device:

  • Saved locations
  • Forecast snapshots and accuracy history
  • Settings and preferences
  • API keys (stored in Keychain)

No data is sent to HogBody Labs servers. There is no cloud sync, no analytics, and no telemetry.

No Accounts

Skeptic does not require or offer account creation. There is no sign-up, no login, and no email collection.

No Tracking or Analytics

Skeptic contains:

  • No analytics SDKs
  • No crash reporting services
  • No advertising frameworks
  • No fingerprinting or device identification

No Ads

Skeptic does not display advertisements and never will.

Third-Party Privacy Policies

The weather services Skeptic contacts have their own privacy policies. When your device fetches forecast data, those services may log the request according to their own policies. Skeptic has no control over how third-party APIs handle requests.

Children's Privacy

Skeptic does not knowingly collect information from children under 13. The app does not collect information from anyone.

Changes to This Policy

If we update this policy, we'll change the date at the top. Since Skeptic doesn't collect data, meaningful changes are unlikely.
